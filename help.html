Dashboard Help & Documentation

About the Dashboard

The Dashboard is a personal web application designed to help you organize and manage your favorite links efficiently. You can create custom sections, add links, and organize them according to your preferences. The application works offline and provides various features to enhance your browsing experience.

Header Icons & Functions

Theme Color: Change the application's primary color theme to personalize your experience.
Dark Mode: Toggle between light and dark mode for comfortable viewing in different lighting conditions.
Change Username: Personalize your dashboard by setting your preferred username.
Retirement Timer: Track the countdown to your retirement date. You can enable/disable and customize the countdown settings.
Import Data: Import your dashboard and todo data from a JSON backup file, or import browser bookmarks from an HTML bookmark export file. Supports both unified backups and legacy formats.
Export All Data: Export all your data (bookmarks, todos, and settings) to a single JSON file for backup or transfer to another device.
Backup Settings: Configure automatic backup schedules and reminders to protect your data with regular backups.
Quick Notes: Open the quick notes feature to create, search, and manage your personal notes. Press Ctrl+Shift+N to quickly access notes from anywhere.
To-do List: Access your to-do list to manage tasks and stay organized.
Undo: Revert your last action (available when applicable).
Clear Storage: Reset the dashboard by clearing all stored data (use with caution).

Main Features

Adding Sections

Use the "Add New Section" form to create custom categories for organizing your links. Enter a section name and click "Add Section" to create a new section.

Adding Links

To add a new link:

  1. Select the target section from the dropdown menu
  2. Enter the link name (how you want it to appear)
  3. Enter the complete URL (must start with http:// or https://)
  4. Click "Add Link" to save

Favorites Section

The Favorites section appears at the top of your dashboard for quick access to your most important links.

Search Functionality

Use the search bar to quickly find links across all sections. The search is performed in real-time as you type.

Command Palette: Press Ctrl+K (or Cmd+K on Mac) from anywhere to open the command palette for quick access to navigation and actions.

Global Search: Press Ctrl+F (or Cmd+F on Mac) from anywhere to open a powerful global search that can find content across both your dashboard links and todo items, even when you're on different pages.

Navigation Shortcuts

Form Shortcuts

Automatic Backup System

Protect your data with the integrated automatic backup system:

Features

  • Scheduled Backups: Set automatic backups to run hourly, daily, weekly, or monthly
  • Smart Reminders: Get notified when it's time to create a backup
  • Easy Configuration: Use the backup settings modal to customize your preferences
  • Test Function: Test your backup system anytime with the "Test Backup Now" button

How to Configure

  1. Click the backup settings icon in the header
  2. Enable automatic backups using the toggle switch
  3. Choose your preferred backup frequency
  4. Optionally enable backup reminders
  5. Click "Save Settings" to apply your preferences

The system will automatically create backups according to your schedule and notify you of successful or failed backup attempts.

Quick Notes

The dashboard includes a quick notes feature for capturing thoughts, ideas, and information on the fly. Access notes instantly via the icon in the header or by pressing Ctrl+Shift+N (or Cmd+Shift+N on Mac).

Creating and Managing Notes

  • Quick Access: Notes are available from both the dashboard and task pages
  • New Note: Click "New" button or use Ctrl+N while in the notes modal
  • Auto-Save: Notes automatically save as you type (within 500ms of stopping)
  • Organize with Tags: Add comma-separated tags to categorize your notes
  • Search: Instantly search notes by title, content, or tags
  • Filter by Tags: Click any tag to filter notes by that tag

Note Features

  • Title and Content: Each note has a customizable title and unlimited content space
  • Markdown Support: Use basic markdown formatting (bold, italic, links, headings, lists)
  • Timestamps: Automatic tracking of creation and modification times
  • Tag Management: Filter and organize notes with a flexible tagging system
  • Sort by Recency: Most recently modified notes appear first

Keyboard Shortcuts

  • Ctrl+Shift+N or Ctrl+` - Open notes modal
  • Ctrl+N - Create new note (when modal is open)
  • Ctrl+S - Manual save (notes auto-save by default)
  • Esc - Close notes modal

Data Backup

Your notes are automatically included in the complete data export (version 2.1+) when you use the export feature. This ensures your notes are backed up along with your bookmarks and tasks.

Task Management System

The dashboard includes a comprehensive enterprise-grade task management system accessible via the icon or by visiting the Task List page.

Getting Started

Click the tasks icon in the header to access the task management system. Tasks are organized into smart views, projects, and can be filtered by tags.

Smart Views

Smart views automatically organize your tasks based on different criteria:

  • My Day: Focus on what matters today - includes overdue tasks, tasks due today, and tasks you've manually added to My Day
  • Inbox: Default location for new tasks - organize them into projects later
  • All Tasks: See all your incomplete tasks in one place
  • Important: High-priority tasks that need attention
  • Upcoming: Tasks due in the next 7 days
  • Completed: Recently completed tasks, sorted by completion date

Creating Tasks

Use the quick add bar at the top of the task list to create new tasks instantly. Just type and press Enter. Tasks are automatically added to the current view or project.

Projects

Projects help you organize tasks into categories:

  • Create Projects: Click "New Project" in the sidebar to create a custom project
  • Customize: Assign custom colors and icons to make projects visually distinct
  • Manage: Edit or delete projects using the buttons that appear in the header when viewing a project (Inbox project cannot be deleted)
  • Organize: Assign tasks to projects from the task detail panel

Task Details

Click any task to open the detail panel and access advanced features:

  • Priority: Set task priority to Low, Medium, or High
  • Due Date: Assign due dates to track deadlines
  • Description: Add detailed notes and context to tasks
  • Project: Move tasks between projects
  • Status: Track progress with Todo, In Progress, Done, or Blocked status
  • My Day: Add tasks to My Day for focused daily planning

Subtasks

Break down complex tasks into manageable steps:

  • Add subtasks from the task detail panel
  • Check off subtasks as you complete them
  • Track overall progress with subtask completion counts
  • Delete subtasks you no longer need

Task Dependencies

Create relationships between tasks to ensure prerequisite work is completed first. Dependencies help you manage complex workflows where some tasks must wait for others to finish.

Adding Dependencies

  • Open the task detail panel for the task you want to block
  • Scroll to the "Blocked by" section
  • Select a task or subtask from the dropdown menu
  • Click the "Add" button to create the dependency
  • The task's status automatically changes to "Blocked"

Dependency Features

  • Task-Level Blocking: Block a task until another entire task is complete
  • Subtask-Level Blocking: Block a task until a specific subtask is complete (subtasks appear indented in the dropdown)
  • Multiple Dependencies: A task can be blocked by multiple tasks and/or subtasks
  • Dependency Badges: Task cards show a ‚õìÔ∏è badge with the count of incomplete blockers
  • Circular Prevention: The system prevents creating dependency loops (e.g., Task A blocks Task B, Task B blocks Task A)

Automatic Status Updates

  • Auto-Unblock: When you complete a blocker task/subtask, all dependent tasks automatically change from "Blocked" to "Todo"
  • Auto-Re-Block: If you uncheck a blocker task/subtask (mark it incomplete), dependent tasks automatically return to "Blocked" status
  • Notifications: Success notifications show which tasks were unblocked or re-blocked
  • Badge Updates: Dependency badges update in real-time as blockers are completed or uncompleted

Viewing Dependencies

  • "Blocked by" Section: Shows what tasks/subtasks are blocking this task from starting, with option to remove dependencies
  • "Blocks" Section: Shows which other tasks are waiting for this task to complete (read-only)
  • Subtask Details: When blocked by a subtask, displays as "Task Name (via: Subtask Name)"
  • Badge Tooltip: Hover over the ‚õìÔ∏è badge to see the names of all incomplete blockers

Managing Dependencies

  • Remove Dependency: Click the X button next to any blocker in the "Blocked by" section
  • Complete Blockers: Complete blocking tasks/subtasks to unblock dependent work
  • Work on Blocked Tasks: You can still edit and add details to blocked tasks, just can't mark them complete until blockers are done

Use Cases

  • Sequential Workflows: Ensure "Review PR" happens before "Deploy to Production"
  • Document Dependencies: Block "Write Report" until "Collect Data" subtask is complete
  • Approval Processes: Block "Launch Feature" until "Get Stakeholder Approval" is done
  • Complex Projects: Map out project phases where later tasks depend on earlier ones

Tags

Use tags for flexible task categorization and filtering:

  • Add Tags: Enter tag names in the task detail panel (separate multiple tags with commas)
  • Filter by Tags: Click any tag to see all tasks with that tag
  • Tag Sidebar: View all tags with task counts in the sidebar Tags section

Pomodoro Timer

The integrated Pomodoro Timer helps you stay focused and productive using the Pomodoro Technique - a time management method that alternates between focused work sessions and short breaks.

Starting a Pomodoro Session

To start a Pomodoro timer for any task:

  • Click the play button on any task card in the task list
  • The timer panel appears in the bottom-right corner of the screen
  • The timer starts automatically with a 25-minute work session (default)

Timer Panel Controls

The timer panel provides several controls to manage your Pomodoro sessions:

  • Pause/Resume: Click to pause the current session or resume after pausing
  • Skip: Jump to the next session (work ‚Üí break or break ‚Üí work)
  • Settings (gear icon): Open settings to customize timer behavior
  • Minimize: Reduce the panel size while keeping the timer running
  • Stop (X button): End the current session completely

How the Pomodoro Cycle Works

The timer follows the classic Pomodoro Technique pattern:

  1. Work Session (25 min): Focus on your task
  2. Short Break (5 min): Take a brief break
  3. Repeat: Complete 4 work sessions with short breaks
  4. Long Break (15 min): After the 4th work session, take a longer break
  5. Cycle Repeats: Start a new cycle of 4 pomodoros

Pomodoro Settings

Click the gear icon in the timer panel to customize your Pomodoro experience:

  • Work Duration: Set work session length (1-60 minutes, default: 25)
  • Short Break: Set short break length (1-30 minutes, default: 5)
  • Long Break: Set long break length (1-60 minutes, default: 15)
  • Pomodoros Until Long Break: Number of work sessions before long break (2-10, default: 4)
  • Enable Sound Notifications: Play a beep sound when sessions complete
  • Auto-start Next Session: Automatically start the next session without clicking Resume

Your settings are automatically saved and persist across browser sessions.

Pomodoro Tracking

The system automatically tracks your completed Pomodoro sessions:

  • Each completed work session increments the pomodoro count for that task
  • The üçÖ tomato emoji badge appears in the task metadata showing the count
  • Pomodoro counts are saved with your task data
  • Track your productivity by seeing how many pomodoros each task requires

Smart Timer Behavior

The Pomodoro timer intelligently responds to task changes:

  • Task Completion: Timer stops automatically when you mark a task as complete
  • Task Deletion: Timer stops if you delete the task it's running for
  • State Persistence: Timer state is saved to localStorage and survives page refreshes (up to 30 minutes)
  • Auto-start Control: Choose whether breaks and work sessions start automatically or require manual resume

Tips for Using the Pomodoro Timer

  • Stay Focused: During work sessions, focus solely on the task at hand
  • Take Breaks: Use break times to rest your eyes, stretch, or grab water
  • Customize Duration: Adjust session lengths to match your attention span and workflow
  • Track Progress: Use the pomodoro count to estimate task complexity and improve future planning
  • Minimize Distractions: The timer panel can be minimized to reduce screen clutter while staying visible

Unified Data Management

The dashboard includes a comprehensive data management system that allows you to:

  • Export All Data: Create a complete backup of your dashboard including bookmarks, todos, retirement settings, and all preferences in a single JSON file.
  • Import All Data: Restore your complete dashboard from a backup file with a single operation.
  • Backward Compatibility: The system supports importing legacy formats (dashboard-only or todo-only backups).
  • Cross-Page Integration: The same backup file works for both the main dashboard and the todo list page.

Regular backups are recommended to prevent data loss. The system will occasionally remind you to export your data if you haven't done so recently.
